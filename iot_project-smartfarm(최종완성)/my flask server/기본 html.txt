어디로 전송할지 지정
<form> 태그 사용하면 됨 (입력 박스들을 태그 침)

<form action="/create/"> create로 전송   (이런식으로 하면 url을 통해서 데이터를 서버로 전송하는 방법을 get방식이라고 함.)
get방식은 동적으로 동작하는 웹서비스에서 특정한 페이지를 식별하는 고유주소로서 사용되는 것. (특정페이지를 읽을때나 쓰자)

새로운 정보를 추가하려는 것이기에 post를 써야함 값을 변경하려고 하기 때문.
즉 <form action="/create/" method="POST"> 이렇게 써야함 (url 변화 없음 + 네트워크의 페이로드에만 나타남.)


또한 플라스크에서 post 방식을 사용하려면
라우트에 methods=['GET', 'POST']을 추가해야함
또한 응답을 확인하고 싶다면 from flask import request를 추가해야함 (응답모듈을 임포트해야함)
이제 함수 호출시 if request.method == 'POST':
	return 'hello'
이제 포스트 방식을 하면 hello 가 뜬다.
만약 title = request.form['title']  body = request.form['body'] 이렇게하고
return title + ',' + body 리퀘스트를 통해 폼에 적어놨던 내용을 다시 불러올 수 있음.
